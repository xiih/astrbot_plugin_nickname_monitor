# 群昵称修改监控插件

## 简介
当群聊中有成员修改自己的群昵称（群名片）时，本插件会自动捕获该事件并发送一条调皮可爱的播报消息，内容包含：
- 修改者的头像
- 修改前的原昵称
- 修改后的新昵称

轻松活跃群氛围，让每次“转职”都充满仪式感！

## 功能说明
- 实时监听群消息中的昵称变更系统提示（由 QQ / Napcat 自动生成）
- 自动提取旧昵称、新昵称以及修改者的 QQ 号
- 通过 QQ 号拼接头像链接，展示头像
- 播报文本风格调皮可爱，可自定义模板
- 仅在群聊中生效，不干扰私聊

## 插件流程
1. 插件加载后，监听所有**群消息**。
2. 当收到一条消息时，用正则表达式匹配是否包含“修改群昵称”关键词。
3. 如果匹配成功，从文本中提取：
   - 原昵称（可能附带 QQ 号）
   - QQ 号（可选，用于获取头像）
   - 新昵称
4. 根据提取到的 QQ 号，拼接公开头像链接 `https://q1.qlogo.cn/g?b=qq&nk={qq}&s=640`。
5. 组装一条包含提示文本和头像的消息，发送到对应的群聊。
6. 停止消息的后续处理（避免干扰其他插件或 LLM）。

## 使用方法
1. 安装插件到 AstrBot 的 `plugins` 目录，或通过插件市场安装。
2. 启动 AstrBot，确保 Napcat（aiocqhttp）适配器已正确连接并正常工作。
3. 无需任何额外配置，插件开箱即用。
4. 当群内有人修改群昵称时，机器人会自动发出播报。

## 配置说明
本插件提供简单配置项，可在 AstrBot WebUI 的“插件配置”中修改：

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `enabled` | bool | `true` | 是否启用昵称修改监控 |
| `custom_texts` |  list | 见示例 | 自定义播报文本，支持占位符 `{old}`、`{new}`、`{qq}`。换行用 `\n` 表示。 |

占位符说明：
- `{old}`：修改前的昵称
- `{new}`：修改后的昵称
- `{qq}`：修改者的 QQ 号（若无则为空）

如果未提取到 QQ 号，头像将不发送，只发送纯文本消息。

## 注意事项
- 该插件依赖 Napcat 适配器上报的群系统消息（类似“XXX 修改群昵称为 XXX”），请确保适配器未过滤此类系统消息。
- 头像链接使用腾讯公开接口，若失效可自行修改代码。
- 正则匹配可能存在误判（极少情况），若昵称本身包含“修改群昵称”字样可能触发，可根据实际情况调整正则表达式。
- 插件会调用 `event.stop_event()` 阻止事件继续传播，避免被其他插件或 LLM 重复处理。
