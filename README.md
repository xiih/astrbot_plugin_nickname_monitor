# 🎭 群昵称修改监控插件

> 当群友偷偷改完群名片，然后发出第一条消息时，机器人会在群内进行**播报** 旧昵称 → 新昵称，并且附带头像

## ✨ 特性

- 不打扰：**只在改昵称的人发送消息后才播报**，避免机器人突然插入无关提示
- 自动识别系统“修改群昵称”提示，提取旧名、新名、QQ号
- 自动拼接头像 `https://q1.qlogo.cn/g?b=qq&nk={qq}&s=640`
- 播报文本可自定义（支持 `{old}`、`{new}`、`{qq}`）
- 仅群聊生效，不干扰私聊

## 🧠 工作机制

1. 群系统提示“XX修改群昵称为YY” → 插件**暂不播报**，只记录事件
2. 等到该成员**后续发送任意消息**时 → 机器人才在群里播报昵称变更
3. 播报包含头像 + 自定义调皮文本，然后清除记录

## 📦 安装

插件市场安装，或手动放入 `astrbot/plugins` 目录后重启astrbot  
需要 Napcat / aiocqhttp 适配器正常上报系统消息。

## ⚙️ 配置（可选）

通过 AstrBot WebUI → 插件配置修改：

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `enabled` | bool | `true` | 是否启用 |
| `custom_texts` | list | 见下文 | 自定义播报文本，换行用 `\n` |

默认文本示例：
```json
["{old} 悄悄改名为 {new} 🐣", "捕获一只转职的 {new}，前身是 {old} ✨"]
